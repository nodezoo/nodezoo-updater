# nodezoo-npm-update

- __Sponsor:__ [nearForm][]

Nodezoo.com micro-service handling npm data.. Please see the [main repo][] for more details.

# Running

```sh
$ node npm-service.js --seneca.options.from=dev.options.js --seneca.log.all
```
